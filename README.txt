AI Desktop Translator 🐱🤖

Минималистичный и быстрый десктопный переводчик документов (DOCX) на базе PyQt6 с возможностью интеграции локальных языковых моделей через API. Проект разработан на стыке лингвистики и Prompt Engineering с использованием искусственного интеллекта для автоматизации рутинных задач переводчика.

✨ Особенности
- Локальный и приватный: Возможность подключения к локальным моделям (например, через Ollama) гарантирует, что ваши документы не утекут в сеть.
- Работа с форматами: Корректная обработка файлов `.docx` с сохранением базовой структуры документа.
- Графический интерфейс: Удобный и понятный GUI, созданный на PyQt6, работающий без необходимости запуска терминала.
- Кроссплатформенность: Код подготовлен для сборки под Linux (Bazzite/Fedora), Windows и macOS.

🛠️ Технологический стек
- Язык: Python 3.11+
- Интерфейс: PyQt6
- Работа с документами: python-docx
- Запросы к API: requests
- Сборка бинарников: PyInstaller

 🚀 Быстрый старт (Локальная сборка для Linux)

Если вы хотите запустить проект из исходного кода или собрать его самостоятельно на системах типа Bazzite/Fedora:

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/Last_Shin/AI_Desktop_Translator.git](https://github.com/ваш_никнейм/AI_Desktop_Translator.git)
   cd AI_Desktop_Translator
2. Создайте виртуальное окружение и установите зависимости:
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
3. Сборка независимого исполняемого файла через PyInstaller:
pyinstaller --noconsole --onefile main.py
Готовый проект появится в папке dist/
